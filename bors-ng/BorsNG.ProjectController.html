    Bors-NG v0.1.0 BorsNG.ProjectController View Source

    Shows a list of repositories, a single repository, and the repository's settings page.

    n.b. We call it a project internally, though it corresponds to a GitHub repository. This is to avoid confusing a GitHub repo with an Ecto repo.
