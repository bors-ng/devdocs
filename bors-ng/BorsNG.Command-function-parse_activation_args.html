    The username part of an activation-by command is defined like this:

    • It may start with whitespace

    • @-signs are stripped

    • ", " is converted to ","

    • Otherwise, whitespace ends it.

      iex> alias BorsNG.Command iex> Command.parse_activation_args("", " this, is, whitespace heavy") "this,is,whitespace" iex> Command.parse_activation_args("", " @this, @has, @ats") "this,has,ats" iex> Command.parse_activation_args("", " trimmed ") "trimmed" iex> Command.parse_activation_args("", "what\never") "what" iex> Command.parse_activation_args("", "") "" iex> Command.parse_activation_args("somebody") [{:activate_by, "somebody"}] iex> Command.parse_activation_args("") [] iex> Command.parse_activation_args(" ") [] iex> Command.parse_activation_args("somebody p=10") [{:set_priority, 10}, {:activate_by, "somebody"}]
